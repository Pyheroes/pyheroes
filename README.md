# Base Telegram Bot Template

This is a generic, production-ready Telegram Bot template built with Flask and Python.

## Features
- **Flask Backend**: Handles webhooks and API endpoints.
- **Local Polling**: Test your bot locally without ngrok/webhooks.
- **Render Ready**: Pre-configured `render.yaml` for one-click deployment.
- **SQLite Persistence**: Configured to use Render Persistent Disks to keep your data safe.
- **Static Frontend**: Includes a base HTML/CSS dashboard for your service.

## Setup
1. Update `db.py` with your Telegram Bot Token.
2. Push to GitHub.
3. Deploy to Render.
